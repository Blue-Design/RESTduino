RESTduino

Ever wanted to interface your software with the physical world but not sure where to start? Familiar with the using REST-style web services? RESTduino might be just the ticket for you!

Load RESTduino on to an Arduino-compatible board coupled with an Ethernet shield and you will be able to interact with the Arduino's input/output pins simply by issuing REST-like URL's to the ipaddress of the Ethernet shield (the repository includes some sample code demonstrating the control of LED's from a JavaScript web app).
